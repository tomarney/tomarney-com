# tomarney<i></i>.com
Source and content for my personal website [tomarney.com](https://tomarney.com), a static website built with [Hugo](https://github.com/gohugoio/hugo) and hosted on [Netlify](https://github.com/netlify). This is my own hand-made theme, but I’m very grateful for the headstart on directory structure from [Niklas Fasching](https://github.com/niklasfasching)’s bare minimum [Whitespace](https://github.com/niklasfasching/whitespace) theme.

If you spot a bug, you're very welcome to create an [issue](https://github.com/t-arney/QPID/issues) or (even better) submit a [pull request](https://github.com/t-arney/QPID/pulls)
